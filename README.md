# 🔥 FireLauncher

FireLauncher is a lightweight and fast launcher for Linux that provides quick access to applications, files, and system tools through a clean and simple interface.

## Features

* ⚡ Fast and lightweight
* 🔍 Instant application search
* 🎨 Clean and modern interface
* 🖥️ Optimized for Linux
* 🛠️ Easy to customize
* 📂 Quick access to files and folders

## Requirements

* Linux
* Python 3.10 or newer
* Git

## Installation

Clone the repository:

```bash
git clone https://github.com/maks1234pro/firelauncher.git
cd firelauncher
```

Install dependencies (if your project includes a `requirements.txt` file):

```bash
pip install -r requirements.txt
```

If you use a virtual environment:

### Linux

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Running FireLauncher

Start the launcher with:

```bash
python3 main.py
```

If your project uses a different entry file, run:

```bash
python3 app.py
```

## Updating

To get the latest version:

```bash
git pull
```
