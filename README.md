# 🔥 FireLauncher

FireLauncher is a lightweight and fast Linux application launcher designed for simplicity, speed, and productivity.

It provides a clean interface and quick access to applications, files, and system tools.

---

## 🚀 Features

- ⚡ Fast and lightweight
- 🔍 Quick search system
- 🎨 Clean and minimal UI
- 🖥️ Linux support
- ⚙️ Easy to use
- 📂 Fast access to apps and files
